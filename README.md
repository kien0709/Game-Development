# Best Education – Launch Game

## Over de game
Dit is een 2D game gemaakt in Unity waarin de speler vallende boeken moet opvangen.

## Hoe start je de game
1. Open Unity Hub
2. Open dit project
3. Open de scene:
Assets/Scenes/SampleScene
4. Klik op de Play knop in Unity

## Besturing
A / ← = links bewegen  
D / → = rechts bewegen

## Doel
Vang genoeg boeken om het level te winnen zonder te veel boeken te missen.
